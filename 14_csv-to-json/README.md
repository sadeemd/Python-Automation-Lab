# Day 14 — Convert CSV to JSON

This lab converts a CSV file into a JSON file using Python.

✅ Why this is useful:
- JSON is widely used in **APIs**
- Useful for **dashboards**, **reports**, and **web applications**
- Easy to read and integrate with other systems

---

## Files
- `report.csv` → Input file (sample data)
- `to_json.py` → Python script to convert CSV to JSON
- `report.json` → Output JSON file (generated)
