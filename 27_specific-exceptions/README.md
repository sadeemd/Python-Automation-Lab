# exceptions_demo.py — Specific Exceptions (ValueError)

## Overview
This mini script shows how to catch a **specific exception** (`ValueError`) instead of using a broad `except:`.

## What it demonstrates
- `try` / `except` flow
- Catching **ValueError** from `int()` when conversion fails
- Printing a clear error message using the exception object `e`

## Files
- `exceptions_demo.py` — main script (no input file needed)

## How to run
```bash
python exceptions_demo.py
