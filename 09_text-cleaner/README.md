# Day 9 — Split + Clean (Text Cleaner)

This mini-lab reads a text file that contains simple `name, age` pairs, cleans extra spaces, and writes a normalized output file.

## Goal
- Split each line by `,`
- Remove extra spaces using `strip()`
- Output clean, consistent lines like: `ali,20`

## Files
- `data.txt` — input (raw text)
- `split_clean.py` — script
- `cleaned.txt` — output (generated)

## Input format (data.txt)
Example:
```txt
  ali , 20
 sara, 30
