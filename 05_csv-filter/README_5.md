# Day 5 — CSV Filter (FAIL Rows)

## Goal
Read a CSV file and create a new CSV file that contains only the rows where `status = FAIL`.

## Files
- `input.csv` — input data
- `csv_filter.py` — script that filters the CSV
- `filtered.csv` — output (generated after running the script)

## Input Format (example)
```csv
item,status
A,OK
B,FAIL
C,OK
D,FAIL
